**Status:** ✅ SHIPPED 2026-05-10

## Overview

Packaged myhelper for reliable cross-platform installation (macOS + WSL/Linux) and made the first-run experience self-guiding from zero to working chat in a single `myhelper setup` command. Added Tavily as the primary search provider alongside SearXNG, with goreleaser automating multi-platform binary releases.

## Milestone Summary

**Key Decisions:**
- SRCH (Phase 29) planned before SETUP (Phase 30) — wizard configures Tavily key, provider must exist to be wired at setup time
- DIST (Phase 28) fully independent — goreleaser/CI does not touch application logic
- Homebrew tap deferred to future requirements (DIST-F01) — curl installer covers WSL primary use case
- OpenAI-compatible endpoint deferred to future requirements (INFER-F01) — Ollama-only for v5.0
- goreleaser v2 schema used (version: 2 header required)
- Wizard uses io.Reader/io.Writer injection for testability — no os.Stdin hardcoding
- mergeHomeConfig uses map-based merge to preserve unrelated config keys
- Hardware detection: nvidia-smi (multi-GPU WR-02 fix: parse first line only) → system_profiler → /proc/meminfo → sysctl fallback
- Package-level HTTP client with 15s timeout (WR-01) to prevent wizard hanging

**Issues Resolved:**
- CR-01/CR-02: pullModel status detection — NDJSON success line detection and error propagation
- WR-01: HTTP timeout missing on wizard HTTP clients
- WR-02: nvidia-smi multi-GPU output — only first line parsed
- WR-03: LoadConfig subtest isolation via t.Chdir(t.TempDir())
- WR-04: MYHELPER_SEARCH_PROVIDER env var added to LoadConfig for explicit provider override

**Issues Deferred:**
- install.sh archive extraction path — wrap_in_directory not explicitly set to false in .goreleaser.yaml; verify on first real tag push
- End-to-end GitHub Actions release — requires first v* tag push to fire (cannot test without real remote)
- Phase 22 spinner test (v3.2 carry-forward) — live TTY test with Ollama+SearXNG

**Technical Debt Incurred:**
