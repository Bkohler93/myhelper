   ```go
   var tokenLimitFlag int
   ```

2. In the existing `func init()` (or add one if absent — root.go currently has no init()), register the persistent flag on rootCmd:
   ```go
   func init() {
       rootCmd.PersistentFlags().IntVar(&tokenLimitFlag, "token-limit", 0, "override token threshold for conversation history (default 4100)")
   }
   ```
   Using 0 as the flag default means "not set" — callers check `if tokenLimitFlag != 0` to apply the override.

3. Export a helper function so command RunE functions can apply the flag override to a resolved config:
   ```go
   // ApplyFlagOverrides applies any CLI flag overrides to a resolved Config.
   // Call this after config.Load() in each command's RunE.
   func ApplyFlagOverrides(cfg *config.Config) {
       if tokenLimitFlag != 0 {
           cfg.TokenThreshold = tokenLimitFlag
       }
   }
   ```

4. Add the import for `"github.com/bkohler93/myhelper/internal/config"` if not already present.
