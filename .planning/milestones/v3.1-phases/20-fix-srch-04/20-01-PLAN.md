### Task 2: Add result_count_present subtest to search_test.go

**File:** `internal/search/search_test.go`

Add a new subtest to the existing `TestSearch_RequestParams` function, after the `pageno_present` subtest:

```go
t.Run("result_count_present", func(t *testing.T) {
    srv := httptest.NewServer(http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        numResults := r.URL.Query().Get("num_results")
        if numResults == "" {
            t.Errorf("expected num_results param in request URL, got empty")
        }
        if numResults != "10" {
            t.Errorf("expected num_results=10, got %q", numResults)
        }
        w.Header().Set("Content-Type", "application/json")
        json.NewEncoder(w).Encode(map[string]any{"results": []map[string]string{}})
    }))
    defer srv.Close()
    cfg := search.Config{Endpoint: srv.URL}
    _, err := search.Search("test", cfg)
    if err != nil {
        t.Fatalf("unexpected error: %v", err)
    }
})
```

**Verify:** `go test ./internal/search/... -run TestSearch_RequestParams/result_count_present -v` passes.
