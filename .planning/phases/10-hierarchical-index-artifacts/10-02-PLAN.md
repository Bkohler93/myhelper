<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| os.Stat result → readIndexFile | stat result drives control flow; local filesystem only |
| ErrStaleFlatIndex → query commands | Error type used as a signal; must not leak to end-user as a confusing message |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-06 | Tampering | cmd/helpers.go readIndexFile path | accept | Path is always constructed as filepath.Join(root, ".myhelper", "project.json") where root is os.Getwd(); no user-controlled path component |
| T-10-07 | Information Disclosure | ErrStaleFlatIndex exposed to user | mitigate | Treat ErrStaleFlatIndex as silent fallback (same as ErrNotExist) in buildInjectedMessages; no error message printed to user |
| T-10-08 | Denial of Service | BuildArtifacts failure blocks init/sync | mitigate | BuildArtifacts errors are propagated (fatal for init/sync); LLM summary failure within BuildArtifacts is non-fatal (logs to stderr, continues) per Pitfall 6 |
</threat_model>
